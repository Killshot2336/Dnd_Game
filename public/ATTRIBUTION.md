# Art & Audio Attribution

## Painted plates (`/public/art`)

Custom-generated fantasy oil illustrations commissioned for Voidline (D&D / Wizards-of-the-Coast–inspired product look: painted, not pixel, not stock photography).

- Environment: room, table felt, GM screen, parchment
- Campaign covers: Ashcrown, Saltwake, Blackroot
- Portraits: class archetypes + Aden / Edward / Jamie

These files are original generated assets for this private table client.
